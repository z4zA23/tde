Projeto 3 - Aplicativo Web

Neste novo projeto, a ideia é criar um aplicativo web simples. Para isso, precisamos compreender quais os conceitos e componentes envolvidos.

Toda aplicação web segue a arquitetura cliente-servidor, onde o cliente é a parte em que o usuário interage, e é executada no navegador (Browser).

No servidor é onde o aplicativo é efetivamente executado, e também onde as informações são armazenadas.

Para o desenvolvimento da parte cliente é preciso de no mínimo três linguagens: HTML, JavaScript e CSS, cada uma com uma função específica.

No HTML o desenho da página é feito, no CSS definimos estilos gráficos para que a página fique mais bonita e organizada. Com JavaScript, podemos fazer a lógica necessária para que a interface tenha um comportamento que facilite a interação do usuário (é possível fazer também algumas verificações sobre o que o usuário digitou como entrada de dados).

Na parte servidor, vamos precisar de duas coisas: um servidor web (Apache) e um banco de dados (MySQL). Tudo isso pode ser fornecido pelo XAMPP.

Depois de instalado, no painel de controle do XAMPP devemos ativar estes dois componentes.

Depois de ativados, podemos rodar a interface para o MySQL chamada de phpMyAdmin. Ela é acessada no navegador com o endereço local da máquina (localhost).

Neste caso, estamos com o cliente e o servidor na mesma máquina, é esquisito, mas é isso mesmo. Para fins de desenvolvimento e estudo, é uma situação bem comum.

Além disso, vale ressaltar que o celular para uso pessoal está presente na vida de 155,2 milhões de brasileiros com 10 anos ou mais. O número representava 84,4% da população a partir dessa idade em 2021, percentual maior que o estimado em 2019 (81,4%). O dado foi revelado nesta sexta-feira pela Pnad Contínua (Pesquisa Nacional por Amostra de Domicílios), coletada pelo IBGE (Instituto Brasileiro de Geografia e Estatística).

Conforme os números, há ainda uma "grande discrepância" no volume de telefones conforme a situação do domicílio. Enquanto 87,1% das pessoas que viviam em área urbana tinham celular para uso pessoal, esse percentual era de 67,6% entre moradores na zona rural.

Essa crescente presença dos dispositivos móveis na vida cotidiana dos brasileiros destaca a relevância de explorarmos o desenvolvimento de aplicativos web que possam atender às necessidades e demandas desse público cada vez mais conectado.

Na passagem de 2019 para 2021, a parcela da população de 10 anos ou mais que tinha acesso à internet por meio dos aparelhos cresceu de 91,7% para 94,8%. Na área rural esse percentual avançou de 80,7% para 89,1%, mas permaneceu menor que o das regiões urbanas, que aumentou de 93% para 95,5%.

Prós e Contras do Uso de Celulares

Prós

  • Conectividade Instantânea: Mantenha-se conectado com amigos e familiares em qualquer lugar e a qualquer momento.
  • Acesso à Informação: Tenha informações educacionais e de entretenimento ao alcance dos seus dedos.
  • Ferramenta de Produtividade: Utilize o celular como uma ferramenta eficaz para trabalho e produtividade.

Contras

  • Fadiga Ocular: A exposição prolongada à tela pode causar fadiga nos olhos.
  • Distúrbios do Sono: A luz azul emitida pelos dispositivos pode interferir no sono.
  • Isolamento Social: A dependência das interações online pode levar ao isolamento social.
  • Impacto Físico: O uso prolongado pode resultar em problemas posturais e dores físicas.

Dicas para um Uso Consciente de Celular

Para manter um equilíbrio saudável, considere as seguintes dicas:

  • Estabeleça Limites: Defina limites de tempo para o uso diário do celular.
  • Evite Uso Antes de Dormir: Reduza a exposição à luz azul evitando o uso do celular antes de dormir.
  • Interações Face a Face: Priorize interações sociais presenciais em vez de virtuais.
  • Exercícios Físicos: Pratique exercícios para mitigar os efeitos físicos do uso excessivo.
